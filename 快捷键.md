# WezTerm 快捷键

本配置已禁用 WezTerm 默认键盘和鼠标绑定，只使用 `config/bindings.lua` 中定义的绑定。

## 修饰键

| 名称 | Windows/Linux | macOS | 用途 |
| --- | --- | --- | --- |
| `SUPER` | `Alt` | `Cmd` | 高频操作，例如新建标签、搜索、调整字体 |
| `SUPER_REV` | `Alt+Ctrl` | `Cmd+Ctrl` | 窗格、标签移动等复合操作 |
| `LEADER` | `Ctrl+Shift+Space` | `Ctrl+Shift+Space` | 进入临时按键模式 |

## 通用功能

| Windows/Linux | macOS | 功能 |
| --- | --- | --- |
| `F1` | `F1` | 进入复制模式 |
| `F2` | `F2` | 打开命令面板 |
| `F3` | `F3` | 打开启动器 |
| `F4` | `F4` | 打开标签导航器 |
| `F11` | `F11` | 切换全屏 |
| `F12` | `F12` | 打开调试覆盖层 |
| `Alt+f` | `Cmd+f` | 搜索当前终端内容 |
| `Alt+q` | `Cmd+q` | 快速选取模式，抓取屏幕上的路径/URL/哈希 |
| `Alt+e` | `Cmd+e` | 打开 emoji/符号选择器，选中后复制到剪贴板 |

## 复制与选择

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl+Shift+c` | 复制到剪贴板 |
| `Ctrl+Shift+v` | 从剪贴板粘贴 |
| `Ctrl+Shift+z` | 向终端程序发送 `Ctrl+z` |
| `Ctrl+Shift+a` | 选择整个回滚区内容 |

## 标签页

| Windows/Linux | macOS | 功能 |
| --- | --- | --- |
| `Alt+t` | `Cmd+t` | 新建默认域标签页 |
| `Alt+Ctrl+n` | `Cmd+Ctrl+n` | 在当前域中新建标签页 |
| `Alt+Ctrl+w` | `Cmd+Ctrl+w` | 关闭当前标签页，不弹确认 |
| `Alt+[` | `Cmd+[` | 切换到上一个标签页 |
| `Alt+]` | `Cmd+]` | 切换到下一个标签页 |
| `Alt+Ctrl+[` | `Cmd+Ctrl+[` | 当前标签页左移 |
| `Alt+Ctrl+]` | `Cmd+Ctrl+]` | 当前标签页右移 |
| `Ctrl+Shift+R` | `Ctrl+Shift+R` | 重命名当前标签页 |

## 窗口与启动器

| Windows/Linux | macOS | 功能 |
| --- | --- | --- |
| `Alt+n` | `Cmd+n` | 新建窗口 |
| `Alt+y` | `Cmd+y` | 输入并执行 `yazi` |
| `F3` | `F3` | 打开 WezTerm 启动器 |

启动器中的 Windows 启动项由 `config/launch.lua` 定义：

| 启动项 | 依赖 |
| --- | --- |
| 打开 Windows PowerShell | `powershell` |
| 打开 PowerShell 7 | `pwsh` |
| 打开命令提示符 | `cmd` |
| 打开 Nushell | `nu`，可选 |
| 打开 Yazi 文件管理器 | `yazi` |
| cxy：以 YOLO 模式启动 Codex | `codex` |
| gwn：创建 Git worktree 并进入目录 | `git` |
| ccd：启动 Claude 并跳过权限确认 | `claude` |
| cxu：更新 Codex 版本 | `pnpm` |
| ccu：更新 Claude 版本 | `claude` |
| pnu：更新 pnpm 版本 | `pnpm` |
| 打开 Git Bash | `D:\software\Git\bin\bash.exe` |

| --- | --- | --- |

| `Alt+Ctrl+\` | `Cmd+Ctrl+\` | 水平分屏 |
| `Alt+Ctrl+-` | `Cmd+Ctrl+-` | 关闭当前窗格，弹出确认 |
| `Alt+w` | `Cmd+w` | 关闭当前窗格，不弹确认 |
| `Ctrl+Shift+w` | `Ctrl+Shift+w` | 关闭当前窗格，不弹确认 |
| `Alt+Ctrl+z` | `Cmd+Ctrl+z` | 切换当前窗格缩放状态 |
| `Alt+Ctrl+k` | `Cmd+Ctrl+k` | 切换到上方窗格 |
| `Alt+Ctrl+j` | `Cmd+Ctrl+j` | 切换到下方窗格 |
| `Alt+Ctrl+h` | `Cmd+Ctrl+h` | 切换到左侧窗格 |
| `Alt+Ctrl+l` | `Cmd+Ctrl+l` | 切换到右侧窗格 |

| `Alt+Ctrl+UpArrow` | `Cmd+Ctrl+UpArrow` | 向上调整窗格大小 |
| `Alt+Ctrl+DownArrow` | `Cmd+Ctrl+DownArrow` | 向下调整窗格大小 |
| `Alt+Ctrl+LeftArrow` | `Cmd+Ctrl+LeftArrow` | 向左调整窗格大小 |
| `Alt+Ctrl+RightArrow` | `Cmd+Ctrl+RightArrow` | 向右调整窗格大小 |

## 字体

| Windows/Linux | macOS | 功能 |
| --- | --- | --- |
| `Alt+UpArrow` | `Cmd+UpArrow` | 增大字体 |
| `Alt+DownArrow` | `Cmd+DownArrow` | 减小字体 |
| `Alt+r` | `Cmd+r` | 重置字体大小 |

## Leader 模式

先按 `Ctrl+Shift+Space`，再按下面的键：

| 按键 | 功能 |
| --- | --- |
| `f` | 进入字体调整模式 |
| `p` | 进入窗格调整模式 |

## 字体调整模式

进入方式：`Ctrl+Shift+Space` 后按 `f`。

| 模式内按键 | 功能 |
| --- | --- |
| `k` | 增大字体 |
| `j` | 减小字体 |
| `r` | 重置字体大小 |
| `Escape` | 退出字体调整模式 |
| `q` | 退出字体调整模式 |

## 窗格调整模式

进入方式：`Ctrl+Shift+Space` 后按 `p`。

| 模式内按键 | 功能 |
| --- | --- |
| `k` | 向上调整窗格大小 |
| `j` | 向下调整窗格大小 |
| `h` | 向左调整窗格大小 |
| `l` | 向右调整窗格大小 |
| `Escape` | 退出窗格调整模式 |
| `q` | 退出窗格调整模式 |

## 鼠标绑定

| 操作 | 修饰键 | 功能 |
| --- | --- | --- |
| 左键单击释放 | `Ctrl` | 打开鼠标下方链接 |
| 左键单击按下 | 无 | 从鼠标位置开始选择单元格文本 |
| 左键单击释放 | 无 | 扩展选择到鼠标位置 |
| 左键拖动 | 无 | 扩展选择到鼠标位置 |
| 左键双击按下 | 无 | 选择单词 |
| 左键双击释放 | 无 | 选择单词 |
| 左键三击按下 | 无 | 选择整行 |
| 左键三击释放 | 无 | 选择整行 |
| 鼠标滚轮上滚 | 无 | 按当前滚轮事件滚动 |
| 鼠标滚轮下滚 | 无 | 按当前滚轮事件滚动 |

## 迁移检查

迁移到新机器后，建议先运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\check-env.ps1
```

如果只想给其他工具读取检测结果，可以导出 JSON：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\check-env.ps1 -Json
```
